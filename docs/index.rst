Material Simulations - TFYA99 documentation
===========================================

This is a material simulations program used to find new materials
with applications in various technical fields.

.. note::

   This project is under active development.

.. note::

   Will be able to simulate materials made out of one chemical element as well as alloys. It will also be able to simulate thin films.

Check out the :doc:`usage` section for further information, including how to
:ref:`install <installation>` the project.
